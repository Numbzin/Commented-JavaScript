🌐 Manipulação do Objeto Location em JavaScript

Vamos explorar como manipular a localização da página (URL) usando o objeto window.location.

🔎 Explicação dos Recursos

window.location.href: A propriedade href permite ler ou alterar a URL completa da página.

window.location.reload(): Recarrega a página atual.

window.history.back(): Volta para a página anterior.

window.history.forward(): Avança para a próxima página, se disponível.

window.location.hash: Define ou obtém a parte da URL após o símbolo #, usado para navegação dentro da mesma página.

📝 Resumo e Dicas

  • Utilize window.location.href para redirecionar o usuário a uma nova página.
  • Use window.location.reload() para atualizar a página, útil para recarregar conteúdo.
  • window.history.back() e window.history.forward() são usados para navegação no histórico do navegador.
  • window.location.hash é útil para navegar dentro da mesma página, como em seções específicas.
